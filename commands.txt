COMMANDS

- Activating venv
	source venv/bin/activate

- Starting llama-server
	./build/bin/llama-server -m ../LLaMAntino-2-chat-13b-hf-UltraChat-ITA.Q2_K.gguf --host 127.0.0.1 --port 8080 --chat-template chatml --cache-reuse 0 --temp 0.5

- Starting the energy_api proxy
	uvicorn energy_api:app --host 127.0.0.1 --port 9000 

- Starting the chatUI application in the ChatUI directory
	nmp run dev
